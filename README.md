# README #

### What is this repository for? ###

* Quick summary

用于集中管理Linux下常用工具的配置文件，方便在一个新的环境中快速建立自己的工作链。

目前包括：emacs(spacemacs),zsh(oh-my-zsh),i3/i3status

* Version

[1.0.0]
